## Getting Started

Follow these steps to get your project up and running.

### Prerequisites

Make sure you have the following software installed:

- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)

### Installation

1. Fork this repository
2. cd dir_name
3. npm install
4. cd client
5. npm install
6. npm run start
